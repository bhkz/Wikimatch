# Marketing Super-Skill

A comprehensive marketing reference that merges the best frameworks, templates, and workflows from **Claude Code marketing skills** and **Perplexity Computer's built-in marketing skills** into a single, unified playbook.

## What's Inside

| Section | Coverage |
|---------|----------|
| **Content Creation & SEO** | 4 blog templates, SEO workflow, social media templates (LinkedIn, Twitter, Instagram, Facebook, TikTok), email templates, video scripts, content repurposing matrix, headline & hook formulas, CTA best practices |
| **Campaign Planning & Demand Gen** | SMART objectives, ICP definition, channel selection (owned/earned/paid), paid media setup guides (LinkedIn, Google, Meta), UTM tracking, budget allocation, content calendars, MQL/SQL handoff, partnerships |
| **Analytics & Performance** | 50+ metrics with benchmarks across email, social, paid, SEO, content, and pipeline; multi-touch attribution models; funnel analysis; reporting templates (weekly, monthly, QBR); A/B testing; forecasting |
| **Competitive Intelligence & PMM** | Research methodology, battlecard template, messaging comparison, narrative analysis, positioning (April Dunford), value propositions, category strategy, content gap analysis, sales enablement, product launch tiers, international expansion |
| **App Store Optimization** | Keyword research, metadata optimization, competitor analysis, A/B testing, app launch workflow |
| **Prompt Engineering for Marketing** | 10 ready-to-use prompt templates, A/B testing for prompts, optimization techniques |
| **Computer-Exclusive Capabilities** | Real-time web research, social monitoring, website deployment, image/video generation, 400+ integrations, scheduled tasks, push notifications, document generation |
